# 🎲 **Digital Dice using Arduino**  

This project is a **fun and interactive Digital Dice** built using an **Arduino**, a **7-segment display**, and a **push button**. Whenever the button is pressed, the system generates a **random number between 1 and 6** and displays it on the **7-segment display**, just like rolling a real dice! 🎲✨  

---

## 🛠️ **Components Used**  
- 🔹 **Arduino Uno** – Microcontroller to generate random numbers and control display.  
- 🔹 **7-Segment Display** – Displays the dice number (1-6).  
- 🔹 **Push Button** – Triggers the dice roll.  
- 🔹 **Resistors & Wires** – Required for circuit connections.  

---

## ⚙️ **How It Works**  
1️⃣ When the **push button is pressed**, the **Arduino generates a random number** between **1 and 6**.  
2️⃣ The generated number is then **displayed on the 7-segment display**.  
3️⃣ The system **waits for the next button press** to generate a new number.  

---

## 📝 **Features**  
✅ **Simulates a real dice roll** 🎲  
✅ **Displays random numbers (1-6) on a 7-segment display**  
✅ **Works with a simple push-button mechanism**  
✅ **Perfect for board games and interactive projects**
